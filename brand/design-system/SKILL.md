Read the `README.md` file within this skill, and explore the other available files
(`styles.css` + `tokens/`, `components/`, `ui_kits/website/`, `assets/`).

If creating visual artifacts (slides, mocks, throwaway prototypes), copy assets out
and create static HTML files for the user to view — link `styles.css` for tokens and
load `_ds_bundle.js` for components. If working on production code, copy assets and
read the rules here to design as an expert in the Light brand.

Brand in one line: **direct, no-nonsense, industrial** — brand red `#be1e2d` + logo
grey `#787c7e`, Montserrat headings, Open Sans body, the feather mark, and a
signature notched corner from the logo banner. Dutch-first copy; "de productie moet
draaien, punt."

If the user invokes this skill without other guidance, ask what they want to build,
ask a few questions, and act as an expert designer who outputs HTML artifacts or
production code as needed.
